# C-Tic-Tac-Toe
More or less smart Tic Tac Toe game using C++

# Program Flow

The board gets saved in a 2-Dimensional array with integers.
0 means empty
1 means Player (X)
2 means PC (O)

The board will be printed every round.
You can enter your field via scanf and it will check instantly if you have won.

The PC will firstly check if it can win, then tries to prevent you from winning or place a random O.
